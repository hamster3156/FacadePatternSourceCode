# このリポジトリについて
Facadeパターンを勉強する時に作成したソースコードです。

# ソースコードについて
| ファイル名 | 役割 |
|-------------|-------------|
| PlayerController | Facadeを利用しているクラス |
| PlayerData | Playerの様々な処理をまとめたFacadeクラス |
| PlayerInputManager | Playerの入力を検知するクラス |
| PlayerPositionManager | Playerを座標変更するクラス |
